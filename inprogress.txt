## 5 HIGH-LEVEL UI STYLE CONCEPTS

1. NEO-BRUTALIST INDUSTRIAL
   - Aesthetic: Raw, bold, deliberately harsh. Maximum contrast.
   - Colors: Jet black (#0a0a0a), Electric yellow (#E8FF00), Hot coral (#FF3D3D), White (#FAFAFA)
   - Typography: "Archivo Black" display / "JetBrains Mono" body
   - Layout: Stacked full-width blocks, thick borders (4px+), no border-radius
   - Key feature: Exposed grid structure, chunky buttons, raw paper-like textures

2. EDITORIAL / MAGAZINE
   - Aesthetic: Sophisticated print editorial. Asymmetric whitespace-heavy.
   - Colors: Warm cream (#FAF7F2), Deep charcoal (#2C2C2C), Olive (#6B7A5E), Burnt sienna (#C4654A)
   - Typography: "Playfair Display" display / "Source Serif 4" body
   - Layout: Multi-column asymmetric, large section headers, elegant dividers
   - Key feature: Magazine-style section breaks, accordion navigation, ruled lines

3. RETRO-FUTURISTIC TERMINAL / CYBERPUNK
   - Aesthetic: CRT terminal meets neon cyberpunk. Glowing and dark.
   - Colors: Deep void (#0D0D12), Phosphor green (#00FF88), Neon cyan (#00E5FF), Soft amber (#FFB800)
   - Typography: "Share Tech Mono" display / "Fira Code" body
   - Layout: Sidebar + window-panel layout with titlebars, ASCII borders
   - Key feature: Scanline overlay, boot-up animation, glowing box shadows, live clock

4. ORGANIC / NATURE-INSPIRED (not implementing — reserved for future)
   - Aesthetic: Earthy, warm, flowing shapes.
   - Colors: Terra cotta, Sage green, Warm ivory, Clay brown
   - Typography: "Fraunces" / "Outfit"
   - Layout: Overlapping organic cards, rounded shapes, flowing sections

5. ART DECO / GEOMETRIC LUXURY (not implementing — reserved for future)
   - Aesthetic: 1920s luxury meets modern UI.
   - Colors: Deep navy, Gold, Cream, Rose
   - Typography: "Poiret One" / "Cormorant Garamond"
   - Layout: Strong angular geometry, decorative borders, symmetric patterns
